### 1. Calculator Web App (Level 2 - Task 1)
A fully functional calculator with a modern, soft-blue UI. It supports basic arithmetic operations, decimal calculations, and includes a history feature.

* **Tech Stack:** HTML, CSS (Grid Layout), JavaScript
* **Key Features:**
    * Responsive 5-column grid layout.
    * History display to track previous inputs.
    * Advanced functions like Square Root (âˆš) and sign toggle (Â±).
    * Error handling for invalid operations.

### 2. Tribute Page - Dr. A.P.J. Abdul Kalam (Level 2 - Task 2)
A responsive tribute page dedicated to the "Missile Man of India." It features a clean, dark-themed design with smooth scroll animations.

* **Tech Stack:** HTML, CSS, AOS Library (Animate On Scroll)
* **Key Features:**
    * **Timeline Section:** Visual journey of key life events.
    * **Book Cards:** Highlight of famous literary works.
    * **Interactive UI:** Hover effects on images and buttons.
    * **Animations:** Elements fade in smoothly as you scroll down.

### 3. To-Do List Web App (Level 2 - Task 3)
A smart task management app that separates pending and completed tasks. It uses LocalStorage to ensure your data persists even after refreshing the page.

* **Tech Stack:** HTML, CSS (Glassmorphism), JavaScript
* **Key Features:**
    * **Data Persistence:** Saves tasks automatically to browser LocalStorage.
    * **Task Separation:** Distinct lists for "Pending" and "Completed" items.
    * **Edit Functionality:** Ability to update existing tasks.
    * **Design:** Modern glassmorphism UI with gradient backgrounds.

### 4. Login Authentication System (Level 2 - Task 4)
A secure authentication simulation that allows users to register, log in, and access a protected dashboard.

* **Tech Stack:** HTML, CSS, JavaScript (LocalStorage)
* **Key Features:**
    * **User Registration:** Stores user credentials securely in LocalStorage.
    * **Authentication Logic:** Verifies email and password matching.
    * **Protected Route:** Prevents access to the Dashboard page without logging in.
    * **Dashboard UI:** A secure area displaying user-specific details.
